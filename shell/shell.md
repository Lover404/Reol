```shell
shell脚本for循环遍历1-100
#!/bin/bash
for i in `seq 1 100`
do
echo $i 
done

循环1-100 in使用和seq
`seq 1 100` 或 {1..100}

#!/bin/bash
for ((i=1; i<=100; i++))
do
echo hh$i 
done
```

```shell
shell变量
1.命名只能用英文字母数字和下划线,不能以数字开头
2.中间不能有空格,可以使用下划线
3.不能使用标点符号
4.不能使用bash的关键字(可用help查看关键字)
5.除了直接赋值还可以用语句赋值
   a=`ls /etc`
   执行语句要加上`	`才可以执行
   for i in $(ls /etc)
6.调用变量加$符号
   a='hi'
   echo $a
   echo ${a}
7.变量的花括号可加可不加,但是在紧挨着变量时不加会找不到变量,会将后面的字符串一起当作为一个不存在的变量无法调用
   a=hh
   echo "wo zhen de shi ${a}hh"
   如果是下面这个样子变量就调用不出来,会识别为ahh变量
   echo "wo zhen de shi $ahh"
8.已经定义的变量可以被重新定义
   #!/bin/bash
   a=hh
   echo $a
   a=enen
   echo $a
   结果为 hh  enen
9.只读变量
   使用readonly可以将变量定义为只读变量,只读变量的值不能被改变
   如果尝试更改只读变量结果会报错
   /bin/sh: NAME: This variable is read only.
10.删除变量
   使用unset命令可以删除变量
   unset a
   删除后不能再次使用,该命令不能删除只读变量
```

```
变量类型
运行shell时，会同时存在三种变量：

1) 局部变量 局部变量在脚本或命令中定义，仅在当前shell实例中有效，其他shell启动的程序不能访问局部变量。
2) 环境变量 所有的程序，包括shell启动的程序，都能访问环境变量，有些程序需要环境变量来保证其正常运行。必要的时候shell脚本也可以定义环境变量。
3) shell变量 shell变量是由shell程序设置的特殊变量。shell变量中有一部分是环境变量，有一部分是局部变量，这些变量保证了shell的正常运行
```

```shell
shell字符串
单引号的限制:
	单引号里的任何字符都会原样输出,单引号字符串中的变量是无效的
	单引号字符串中不能出现单独一个的单引号,使用转义符也不行,但可成对出现，作为字符串拼接使用
	
双引号:
	双引号里可以有变量
	也可以出现转义字符
	your_name='runoob'
	str="Hello, I know you are \"$your_name\"! \n"
	echo -e $str

获取字符串长度
	a='1234'
	echo ${#a}
	
截取字符串
	a='can I go where you go?'
	echo ${a:2:6}
	从第三个字符开始截取,截取六个字符

查找字符串
	a='can I go where you go?'
	echo `expr index "$a" Iw`  #输出 5
	用反引号查找,变量要加双引号,查找I或者w的位置,哪个字母先出现就输出哪个字母的位置
```

```shell
shell数组
在shell中用括号来表示数组,数组元素用空格隔开
数组名=(元素1 元素2 ... 元素n)
数组名[0]=元素0
数组名[1]=元素1
数组名[n]=元素n
可以不使用连续的下表,而且下标的范围没有限制

读取数组元素的一般格式是
${数组名[下标]}
@可以获取数组中的所有元素
${数组名[@]}

获取数组的长度(井号#)
变量名=${#数组名[@]}
变量名=${#数组名[*]}
获取数组单个元素的长度
变量名=${#数组名[元素下标]}
```
